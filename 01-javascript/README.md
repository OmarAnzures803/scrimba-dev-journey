Learn JavaScript – Curso completado
Durante este curso trabajé más de 140 retos prácticos para aprender los fundamentos de JavaScript. Empecé con ejercicios básicos de variables, funciones y DOM, y fui aplicando lo aprendido directamente en proyectos.

Construí una aplicación de contador de pasajeros para practicar manipulación del DOM, luego desarrollé el juego Blackjack para afianzar condicionales, arrays y bucles, y finalmente creé una extensión para Chrome, integrando eventos y lógica de interacción con el navegador.

Al terminar, adquirí experiencia práctica en conceptos clave como funciones, parámetros, operadores lógicos y de comparación, manejo de arrays y objetos, eventos con addEventListener, y métodos como getElementById, innerText y textContent.
